# GPP-Bogo-v-Boltz

This repository contains 

- Julia code for the code Boltzmann, Bogoliubov methods (both numerical and using stationary point approximation).
- Mathematica notebook for the matrix element calculation.
    - matrixelement.nb contains all the necessary results.
    - matrixelement_no-feyngrav.nb contains the calculation for the second diagram without feyngrav.

## Associated paper 

Graviton Production from Inflaton Condensate: Boltzmann vs Bogoliubov: [2604.12687](https://arxiv.org/abs/2604.12687)
